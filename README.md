# Python based LBO,DCF & Multiples Valuation Simulator
This prject is an interactive valuation simulator that helps users explore how financial assumptions impact investment returns. It includes LBO (leveraged buyout), DCF (Discounted Cash Flow), and Multiples models all accessible through a browser interface. 
Users can modify key inputs such as leverage, growth rate, exit multiples, or discount rate using sliders and instantly visualize changes in enterprise value, equity value, MOIC, and IRR.
